BIO

20 y.o born in Suriname. A freelance developer

passionate about creating interactive web

experiences and game mechanics. I enjoy

blending creativity with technical problem-solving.

Background

My first exposure to HTML came when I tried to

“hack” Roblox by changing values in the inspect

tool, only to realize it was just a visual trick. That

curiosity led me to explore how websites are built,

and soon I was learning HTML, CSS, and JavaScript

with friends. More recently, I’ve experimented with

C# and Unity while helping a friend in game

development, which deepened my interest in

programming and web development.
